Garden PingPong

In the garden, use PingPong to eliminate insects and fallen leaves, but be careful of sudden gusts of wind!

Project Description

Garden PingPong is an innovative garden-themed game where players use a ping pong paddle to eliminate insects and fallen leaves in the garden. The game features a wind system that adds challenge and fun to the gameplay.

Key Features

  • Garden-themed game environment
  • Ping pong paddle control system
  • Diverse targets (insects and fallen leaves)
  • Dynamic wind system
  • Smooth physics engine
HTML5 CSS3 JavaScript Canvas API Web Audio API

This project demonstrates my professional skills in game development and physics engine design, especially in creating fun and challenging game mechanics.
